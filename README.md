House Price Prediction System 
• Developed a modular machine learning pipeline for house price prediction using feature engineering, data preprocessing,
and regression modeling.
• Built automated Scikit-learn Pipelines with ColumnTransformer for scalable training, validation, and model evaluation (R2,
RMSE, MAE).
• Integrated MLflow for experiment tracking, model versioning, and performance monitoring to improve reproducibility and
deployment readiness.
Tech Stack: Python, Scikit-learn, Pandas, NumPy, MLflow, Matplotlib
